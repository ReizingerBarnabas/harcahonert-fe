# Harc a honért frontend 
Területhódító Kvízjáték (Fejlesztés alatt)
A játék három személlyel játszható. Célja, hogy a játékos helyesen válaszoljon a kérdésekre, ezáltal magyarország egyes megyéit elfoglalva. A játékot az nyeri, aki a legtöbb területtel rendelkezik.

Territory Conquest Quiz (Work in Progress)
A 3-player strategic quiz game. Players compete to conquer Hungarian counties by answering questions correctly. The winner is the player who controls the largest territory. This project focuses on competitive multiplayer logic and map-based visualization.
